🔬 QUANTA v11.5b — Advanced Technology Stack

Every prediction in QUANTA is backed by peer-reviewed research. Below is the complete list of mathematical models, algorithms, and techniques powering the system.

🎯

Triple Barrier Method

Labeling Engine

Replaces naive endpoint returns with proper barrier-crossing labels. A trade is labeled WIN if price hits Take-Profit before Stop-Loss within the time window; LOSS if SL hits first; TIMEOUT otherwise.

Label(t) = { 1 (WIN) if ∃ τ ∈ [t, t+T] : P(τ) ≥ P(t) × (1 + TP) 0 (LOSS) if ∃ τ ∈ [t, t+T] : P(τ) ≤ P(t) × (1 − SL) skip if neither barrier hit within T bars } TP = k₁ × ATR(t), SL = k₂ × ATR(t)
📚 López de Prado, "Advances in Financial ML" (2018), Ch. 3
📊

Brier Score Calibration

Probability Quality Metric

Measures how well each specialist's predicted probabilities match actual outcomes. Used to dynamically reweight the ensemble — well-calibrated models get more influence.

Brier = (1/N) Σᵢ (pᵢ − oᵢ)² where pᵢ = predicted probability oᵢ = actual outcome (0 or 1) Perfect = 0.0 | Random = 0.25 | Worst = 1.0
📚 Brier, G.W. "Verification of Forecasts" (1950), Monthly Weather Review
🎲

Kelly Criterion

Position Sizing

Calculates the mathematically optimal fraction of capital to risk per trade, maximizing long-term geometric growth while preventing ruin. QUANTA uses Half-Kelly for safety.

f* = p − (q / b) where p = win probability (from ensemble) q = 1 − p (loss probability) b = TP/SL ratio (reward-to-risk) Half-Kelly: risk% = 0.5 × f* Clamped to [0.3%, 2.5%] of equity
📚 Kelly, J.L. "A New Interpretation of Information Rate" (1956), Bell System Technical Journal
🔀

Hidden Markov Model (HMM)

Market Regime Detection

Unsupervised model detecting latent market states (Bull / Range / Bear) from returns, volatility, and trend strength. Optimal K found via BIC minimization. Routes predictions to regime-specialized agents.

States: S = {s₁, ..., sₖ} (K found via BIC) Observations: Xₜ = [log_return, ATR%, ADX] Transition: A[i,j] = P(sₜ₊₁=j | sₜ=i) Emission: B[j](x) = N(x; μⱼ, Σⱼ) BIC(K) = −2·ln(L) + k·ln(n) Decode path: Viterbi algorithm
📚 Baum-Welch (1970); BIC selection — Schwarz (1978)
🤖

PPO Neural Network

Deep Reinforcement Learning Gate

Proximal Policy Optimization agent that learns to approve or veto trades based on historical outcomes. Uses clipped surrogate objective to prevent catastrophic policy updates.

L_CLIP(θ) = E[min( rₜ(θ)·Âₜ, clip(rₜ(θ), 1−ε, 1+ε)·Âₜ )] rₜ(θ) = π_θ(aₜ|sₜ) / π_θold(aₜ|sₜ) ε = 0.2 (clip range) Reward: Differential Sharpe Ratio ΔSₜ = (Bₜ₋₁·Rₜ − Aₜ₋₁) / (Bₜ₋₁ − Aₜ₋₁²)^(3/2)
📚 Schulman et al. "PPO Algorithms" (2017); Moody & Saffall (2001)
🌳

CatBoost Gradient Boosting

Core Prediction Engine

Each Greek specialist is a CatBoost classifier with domain-specific feature masks. Uses ordered boosting to prevent target leakage and GPU acceleration for real-time inference.

F(x) = Σₘ γₘ · hₘ(x) (additive trees) Loss: Logloss = −[y·log(p) + (1−y)·log(1−p)] Regularization: L₂ leaf penalty (λ = 5.0) Sampling: Bernoulli subsample = 0.8 7 specialists × 800–1500 iterations each GPU inference: ~5ms per prediction
📚 Prokhorenkova et al. "CatBoost" (2018), NeurIPS
⚖️

Entropy-Regularized Ensemble

Weight Optimization

Ensemble weights are optimized using a blend of Brier quality and validation AUC, with Shannon entropy regularization to prevent any single model from dominating.

w* = argmax Σᵢ scoreᵢ·wᵢ + λ·H(w) H(w) = −Σᵢ wᵢ·ln(wᵢ) (Shannon entropy) λ = 0.3 (regularization strength) Constraints: wᵢ ∈ [0.05, 0.25] (no single model > 25%) Σ wᵢ = 1 score = 0.4·AUC + 0.6·(1 − Brier)
📚 Krogh & Vedelsby (1994); REPO — Markowitz + Shannon
📈

VPIN Order Flow

Volume-Synchronized Probability of Informed Trading

Estimates the probability that current volume is driven by informed traders rather than noise. High VPIN precedes large moves and is used by flow-specialist agents.

VPIN = Σ|V_buy − V_sell| / (n · V_bucket) Buy/Sell classification: Bulk Volume (BVC) V_buy = V · Φ(ΔP / σ_ΔP) V_sell = V − V_buy Alert threshold: VPIN > 2σ above rolling mean
📚 Easley, López de Prado, O'Hara (2012), Journal of Portfolio Mgmt
🔢

Fractional Differencing

Stationarity Preservation

Makes price series stationary while preserving long memory — unlike integer differencing which destroys predictive information. Used as features for mean-reversion detection.

d ∈ (0, 1) — fractional order (1−B)^d = Σₖ C(d,k)·(−B)^k C(d,k) = Γ(d+1) / [Γ(k+1)·Γ(d−k+1)] d ≈ 0.4: minimum d for ADF stationarity while retaining 95%+ correlation
📚 Hosking (1981); López de Prado (2018), Ch. 5
🧬

Temporal Fusion Transformer (TFT)

Deep Learning Sequence Model

Attention-based neural network that processes multi-horizon temporal data. Uses variable selection networks to automatically identify the most relevant features per time step.

Architecture: Input → Variable Selection → LSTM Encoder → Multi-Head Attention (4 heads) → Gated Residual Network → Output Params: ~64 hidden, dropout=0.1 Optimizer: Adam (lr=1e-3) Loss: BCE with logits
📚 Lim et al. "Temporal Fusion Transformers" (2021), Int. J. Forecasting
🧊

Hurst Exponent & Sample Entropy

Market Microstructure Analysis

Hurst exponent detects trending (H>0.5) vs mean-reverting (H<0.5) behavior. Sample entropy measures market complexity — low entropy signals predictable regimes.

Hurst (R/S method): H = log(R/S) / log(n) H > 0.5 → trending | H < 0.5 → mean-reverting Sample Entropy: SampEn = −ln(A/B) where A = template matches at m+1 B = template matches at m
📚 Hurst (1951); Richman & Moorman (2000)
💰

Kyle's Lambda & Amihud Illiquidity

Market Impact Estimation

Kyle's Lambda measures price impact per unit of volume — high λ means low liquidity. Amihud ratio captures absolute return per dollar of volume, identifying illiquid conditions where slippage is dangerous.

Kyle's Lambda: λ = Cov(ΔP, V) / Var(V) Amihud Illiquidity: ILLIQ = (1/N) Σ |Rₜ| / Volumeₜ Both normalized to z-scores for feature input
📚 Kyle (1985), Econometrica; Amihud (2002), J. Financial Markets
📉

Black-Scholes Barrier Probability

Theoretical Win Rate Estimation

Adapted from options pricing theory, computes the theoretical probability that a barrier (TP/SL) will be hit given current volatility. Provides a model-free baseline for comparison with ML predictions.

Kou Double-Exponential Barrier: P(hit TP) ≈ exp(−2·μ·TP / σ²) BS Time Decay: θ(t) = −S·σ·N'(d₁) / (2√T) Implied Vol Ratio: IVR = σ_realized / σ_historical
📚 Black & Scholes (1973); Kou (2002), Management Science
🔄

CPCV Walk-Forward Validation

Combinatorial Purged Cross-Validation

Prevents look-ahead bias in time-series by purging overlap between train/test sets and using combinatorial paths. Model registry gates deployment: only models that beat previous generation go live.

For each fold k: Train: data[0 : t_k − purge_window] Test: data[t_k : t_k+1] Embargo: gap of h bars between sets Deploy gate: AUC_new > AUC_prev − tolerance Brier_new < 0.30 (max threshold) Accuracy improvement > 1%
📚 López de Prado (2018), Ch. 7 — "Cross-Validation in Finance"
